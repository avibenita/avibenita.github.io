1. Include the Module

<iframe src="DataInputModule.html" width="100%" height="600"></iframe>

2. Listen for Data Events

// Listen for data loaded events
window.addEventListener('message', function(event) {
    if (event.data.type === 'dataLoaded') {
        const { data, headers, rowCount, columnCount } = event.data.payload;
        console.log('Data loaded:', data);
        // Process your data here
    }
});

// Or access the module directly (if same-origin)
const moduleWindow = document.getElementById('dataInputFrame').contentWindow;
const data = moduleWindow.DataInputModule.getData();

3. API Methods

  • getData() - Returns the parsed data array
  • getHeaders() - Returns the column headers array
  • getDataInfo() - Returns complete data information object
  • hasData() - Returns boolean indicating if data is loaded
  • setData(data, headers) - Set data programmatically
  • clear() - Clear all data
  • loadDemo() - Load demo data
