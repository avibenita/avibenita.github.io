Updating linked items

 

EzPaste enables to paste charts and tables as linked objects attached to the workbook from which they originate

The problem with Excel is that changing the items in the workbook don’t update them automatically in PowerPoint so that you need to issue manually an “update link” for each item

EzPaste enables to update automatically all the linked items in PowerPoint in 2 modes:

·       Updating at once al the linked items in the active presentation

·       Refresh continuously the linked items so that any changes in the workbook will be automatically reflected in the active presentation as long as the active workbook and active presentation are opened
