Paste to predefined slides in PowerPoint

 

EzPaste advantages are obvious when you have to prepare the same presentation again and again based on the same Excel file where only the data changes. With a click of a button, all the work is done!

In its default mode, EzPaste copy the Excel selected items to automatically added new slides at the end of the active presentation file     

But you may choose to paste the selected items to certain fixed slides in an existing presentation file which acts as a template

You do so by selecting the proper pasting mode and provide the slides numbers where you want the Excel objects to be pasted

All the three pasting formats are available

Only items in the form with a slide number and a positive check are pasted to PowerPoint

Existing elements in the concerned slides are unaffected

Excel file "remembers" the slides numbers when it is reopened
