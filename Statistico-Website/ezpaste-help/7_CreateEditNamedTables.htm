Create/Edit Named Tables

 

EzPaste provides a built in utility for managing Tables of data in Excel

A table of data is simply a rectangular range within a worksheet for which a specific name is assigned. The name serves as a fixed reference address for that table, regardless of the data inside the table. The utility is similar to the Naming Excel procedure ("Insert" Menu in Office 2003, "Formula" in Office2007 and up) and based on it. The Excel Naming mechanism and the EzPaste utility provide the same functionality

For Example, if we name this table as "MyTable", this name will always refer to the range address A1:D6, regardless of the data inside the table

Tip: You may define a range that contains many objects (for example a table and 2 charts) so that all the objects will appear on the same slide

 

 

Creating names with this utility is totally equivalent as doing it with Excel Names Manager built in mechanism. You may prefer the Excel way…The easiest and quickest way for defining a name is by selecting a range and entering its name in the small window about the column A header, then press Enter
