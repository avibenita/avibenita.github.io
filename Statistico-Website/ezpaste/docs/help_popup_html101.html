Quick tip: Only checked items are exported when you click Continue.

Detected item types

EzPaste scans your workbook and recognizes:

  • Chart sheets — full-sheet charts on their own worksheets.
  • Embedded charts — charts placed on regular worksheets.
  • Named tables — Excel tables you have defined in the workbook.
  • Named ranges — named cell ranges treated like tables for export.

Table columns explained

  • Name / Type — the Excel name and object type.
  • Worksheet — the sheet where the item is located.
  • Order — export sequence for checked items (1, 2, 3…). Use Up/Down to move rows. Checked items export from top to bottom in the list.
  • Caption — your label for the item. It can appear as a slide title, Word caption, or file name.
  • Slide # — PowerPoint only, in Predefined Slides mode. Enter the slide number for each checked item.

Slide modes: Choose New Slides to add a new slide for each item. Choose Predefined Slides to paste into slide numbers you specify.

Saved settings: Selections, captions, order, and slide numbers are stored in a hidden sheet in your workbook and restored the next time you open the file.
