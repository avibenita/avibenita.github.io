EzPaste can export multiple Excel items at once to five destinations. Choose the one that fits your workflow.

📊PowerPoint DEFAULT

Inserts each selected item into the active PowerPoint presentation, usually on new slides at the end of the deck.

  • New slides: one slide per checked item.
  • Predefined slides: paste into slide numbers you choose.
  • Captions: item captions can become slide titles.
  • Layout options: size, position, and style settings appear after you click Continue.

A PowerPoint presentation must be open before you export.

📄Word

Inserts each item into the active Word document, typically one item per page.

  • Additional formatting options are available after you click Continue.
  • Excel formatting is preserved as closely as possible.

A Word document must be open before you export.

📋PDF

Creates a PDF file with one page per selected item.

  • Standalone PDF output — no extra printer setup required.
  • The PDF opens automatically when export finishes.
🌐HTML

Creates a single HTML file with a section for each selected item.

  • Easy to share and view in any web browser.
  • The HTML file opens automatically when export finishes.
🖼️Image files

Creates one image file per selected item in a folder you choose.

  • Supports PNG, JPG, GIF, BMP, and TIFF.
  • The output folder opens automatically when export finishes.

Tip: Use PowerPoint for recurring presentations. Use PDF or HTML when recipients may not have Microsoft Office.
