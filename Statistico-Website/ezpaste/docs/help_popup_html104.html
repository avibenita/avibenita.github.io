PowerPoint exports can use one of two slide insertion modes.

Automatic new slides (default)

EzPaste adds a new slide for each checked item in export order.

  • One item per slide.
  • Slides are created in list order for checked rows.
  • No slide numbers are required.
  • Best for building a presentation from Excel data.

Recurring reports: When the same Excel file feeds the same presentation layout, you can refresh the deck with one click after the data changes.

Predefined slides (template mode)

Paste checked items into specific slide numbers in an existing presentation template.

How to use template mode

1Select Predefined Slides.

2Enter a slide number for each checked item in the Slide # column.

3Click Continue to paste into those slides.

Important rules for predefined slides

Only checked items with a valid slide number are pasted. Unchecked rows and rows without a slide number are skipped.

Safe template processing

  • Slides that are not targeted remain unchanged.
  • Only the specified slides receive new content.
  • More than one item can be assigned to the same slide number.

When to use each mode

Use automatic new slides when:
  • You are creating a new presentation from Excel.
  • You want one item per slide without planning slide numbers.
Use predefined slides when:
  • You already have a PowerPoint template with fixed layouts.
  • You need items on specific slide numbers such as 3, 7, or 12.
  • You update the same report layout on a regular schedule.
