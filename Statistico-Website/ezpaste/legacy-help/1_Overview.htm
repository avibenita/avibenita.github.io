Overview

 

EzPaste (to be pronounced "Easy Paste"), is  intended for completely automating and managing the otherwise manual process of copying and pasting objects (charts and tables) from Excel to PowerPoint and 5 other popular destinations:-

 

 

EzPaste can handle three types of Excel objects:-

Embedded charts in worksheets

Charts as separate sheets

Data ranges in worksheets

 

 

 

EzPaste main attributes

Batch pasting mode (many items at once) and Selection swift pasting

Unlimited number of Excel charts and tables may be pasted AT ONCE

Complete control over the items to paste from Excel

Complete control over the pasting order

2 operating modes while pasting to PowerPoint:-

To new slides automatically added to the active PowerPoint file

To predefined slide numbers in the active PowerPoint file

Automatic updating of linked charts and tables in PowerPoint

Complete control over the positioning of the pasted objects on the PowerPoint slide

Assigning styles (shadow, reflection...) to pasted objects as picture (in Office 2007 and up)

Assigning object title directly from EzPaste, to be reused each time

Built-in utility for creating and editing data-range names to be pasted as tables

All the EzPaste settings of a specific Excel file are saved within the file and automatically retrieved once the file is reopened

Each selected object could be previewed directly in the EzPaste main form

 

 

Unregistered version limitations

Pasting many items at once in PowerPoint and Word is disabled after 10 days or a certain number of uses. Up to 10 Excel items may be pasted at once. Some nagging screens

The unregistered version should  be uninstalled before installing the registered one
