EzPaste

“Easy Paste” automates and controls bulk copy-paste of Excel objects—charts, tables, ranges—into PowerPoint, Word, PDF, HTML or images. :contentReference[oaicite:9]{index=9}

What Is EzPaste?

EzPaste is a lightweight Excel add-in (no external dependencies) that lets you select up to unlimited charts and tables and paste them all at once in one click :contentReference[oaicite:10]{index=10}.

Whether your data lives as embedded charts, chart sheets, or named tables, EzPaste consolidates the tedious copy-paste work into a single, repeatable operation.

What Is It For?

  • Batch-export dozens of charts/tables into new or existing slides in PowerPoint :contentReference[oaicite:11]{index=11}
  • Insert images, editable shapes, embedded or linked Excel objects into Word, PDF, HTML, or image files
  • Automatically update presentations from a template when the source Excel workbook changes

Key Strengths

Zero-Config Persistence

All your selections, captions, ordering & modes are stored silently in a hidden sheet—reopen your workbook and settings load automatically :contentReference[oaicite:12]{index=12}.

Precision Layout

Two PowerPoint modes:
• New slides auto-added
• Paste into predefined slide numbers
Full control over position, size & style

Flexible Output Formats

Choose from:
• High-fidelity images (EMF/PNG)
• Native Office shapes
• Embedded worksheets
• Linked Excel objects :contentReference[oaicite:13]{index=13}

Seamless Installation

EzPaste is a single `.xlam` add-in—no COM registrations, no external libraries. Double-click to install in Excel’s Add-Ins folder.

Quick Install

1. Download EzPaste.xlam to %AppData%\\Microsoft\\AddIns
2. In Excel: File → Options → Add-Ins → Go → Browse… → select EzPaste.xlam
3. Enable in the Add-Ins dialog and restart Excel
