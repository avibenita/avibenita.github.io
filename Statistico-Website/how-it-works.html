Getting Statistico into Excel

Statistico is an Office Add-in — a lightweight web application that runs inside Excel through Microsoft's official add-in framework. No software to install locally.

Microsoft AppSource

Find Statistico in the Excel Add-ins store (Insert → Get Add-ins). One click installs it and it appears in your ribbon immediately.

IT / Admin deployment

Organisations can deploy Statistico centrally via Microsoft 365 Admin Center — it appears automatically for all users, no individual action needed.

Sideload for evaluation

Developers and evaluators can sideload a manifest file directly into Excel to test Statistico in any environment without a store listing.

Once installed, Statistico lives in your Excel ribbon. The analysis engine runs entirely in the cloud — nothing is installed on your machine. Results and session state are written back into your workbook, so Excel remembers your model and settings even after closing. Results are regenerated instantly when you reopen. Your original data and existing sheet content are never modified.

Security & privacy

Statistico is built on Microsoft's Office Add-in platform — the same security model that governs every enterprise Excel extension worldwide.

Data never leaves Excel unencrypted

All communication between the add-in and the cloud engine uses TLS encryption. Your data is transmitted securely and is never stored on our servers after the session ends.

Microsoft sandboxed environment

Office Add-ins run in an isolated browser sandbox inside Excel. Statistico cannot access other files, system resources, or any data outside the workbook you explicitly open it on.

No account required to start

You can explore Statistico without creating an account. No personal data is collected during a standard analysis session.

Enterprise-ready deployment

IT administrators can deploy and control Statistico centrally via Microsoft 365 Admin Center, with the same governance policies applied to all other Office Add-ins in your organisation.

Stateless cloud processing

Computation requests are stateless — each analysis call is independent. No session data, no identifiable information, is retained in the cloud after a result is returned.

Results stay in your workbook

Your model configuration is written back into a dedicated area of your Excel workbook — results regenerate instantly when you reopen it. Your original data and existing sheet content are never modified.

Statistico is distributed through Microsoft AppSource, which requires every add-in to pass Microsoft's security and privacy review before publication. You are protected by the same standards as any Microsoft-certified Office extension.

Statistico runs entirely in the cloud through an Excel task pane — nothing installs on your machine. Your data stays in Excel, analysis happens in the cloud, and your model is saved back into the workbook so results regenerate instantly next time you open it — without touching your original data or any existing sheet content.
