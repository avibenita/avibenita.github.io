How the Statistico Calculators Hub works

1) Pick a calculator family

Start from the family hub and choose the functional area relevant to your decision task.

  • Distributions, Power, Staffing, QC, and Effect Size
  • Organized for execution-first workflows

2) Launch directly into tools

Open the launcher/workspace and run calculators immediately with minimal setup overhead.

  • Designed for fast input-to-output cycles
  • Website launch is browser mode only
  • Office.js mode is installed via Microsoft AppSource in Excel

3) Execute and iterate

Set parameters, inspect charts/tables, and iterate quickly to compare scenarios.

  • Immediate feedback across calculator families
  • Practical what-if loops for decision support

4) Move to action

Use computed outputs as operational inputs for planning, reporting, and implementation.

  • Execution-focused calculators for applied outcomes
  • Built to reduce time from question to decision

Launch in browser + install Office.js from AppSource

From this website, launch is browser mode only. For Excel Office.js mode, install Statistico™ from the Microsoft AppSource/Add-ins store.

Install Office.js mode (Excel)

Install from AppSource in Excel:

  1. Open Excel and go to Insert ? Get Add-ins.
  2. Search for Statistico™.
  3. Add it, then open it from the ribbon/taskpane.
