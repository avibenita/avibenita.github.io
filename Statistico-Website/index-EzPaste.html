EzPaste FAQ

Answers from the EzPaste MasterHelp documentation.

What can EzPaste do for me?
EzPaste automates copying and pasting charts and tables from Excel to PowerPoint, Word, PDF, HTML, and picture files. It identifies objects in the active workbook automatically — paste one item or hundreds with a single run.
Can I paste only my current Excel selection?
Yes. Use selection paste from the EzPaste toolbar to send the active chart or table in one step — instead of copy → switch app → paste special. Available for PowerPoint, Word, and picture files.
Which Excel objects does EzPaste recognize?
  • Embedded charts in worksheets
  • Charts on separate chart sheets
  • Tables (rectangular data ranges and named tables)
In what format are objects pasted?
  • Flat pictures (with optional styles such as shadow and reflection)
  • Editable Excel objects in PowerPoint or Word
  • Linked objects that update when the source workbook changes
How do I see which objects EzPaste found?
The main EzPaste form lists every detected chart and table. Select an item to preview it, and check or uncheck items to control what will be pasted.
Do I redefine settings every time I reopen the workbook?
No. EzPaste saves all settings inside the Excel file. Reopen the workbook and your configuration is restored automatically — ideal for monthly reports you run again and again.
How do I access EzPaste in Excel?
Install EzPaste as a regular Excel add-in. After installation, an EzPaste tab appears in the Excel ribbon.
How does EzPaste know where to paste?
EzPaste monitors open and active destination files. For PowerPoint you can paste to new slides added automatically, or to predefined slide numbers. Before pasting, EzPaste shows how many objects will go where and asks for confirmation.
Is there a limit to how many objects I can paste?
No — with a licensed copy you can paste unlimited objects in one run.
What else can appear on a slide besides the chart or table?
You can add a slide title (entered in EzPaste and reused each run) and the path of the source Excel sheet.
Can several objects go onto the same slide?
Yes. Build the layout you want on a worksheet, define the range as a named object, and EzPaste treats it as a single paste item.
How is paste order controlled?
Objects paste in the order shown in the main list. Sort by appearance in Excel, object name, or sheet name — or move items up and down manually.
How does EzPaste identify tables for pasting?
EzPaste includes a built-in utility for creating and naming data tables. Only named tables are identified — the name stays a fixed reference even when the data changes.
How is object positioning on PowerPoint slides controlled?
EzPaste provides a visual layout utility so you can set exact positioning and sizing on target slides.
Why does pasting sometimes take longer?
Pasting as editable Excel objects takes longer than picture mode because Office transfers additional spreadsheet metadata with each object. Switch to picture format for faster runs when editability is not required.
What license is provided?
A lifetime license for one computer ($49). Transfer to a new machine via deactivate/reactivate. Multi-seat licensing is available for teams — contact us.
What is the difference between trial and registered versions?
The trial lets you evaluate the workflow with limits: batch pasting to PowerPoint and Word is restricted after a short period, up to 10 items at once, and occasional reminder screens. The registered version removes those limits.
