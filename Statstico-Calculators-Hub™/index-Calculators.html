Continuous Distributions

Interactive calculators for continuous probability distributions:

  • Normal Distribution - Z-scores, probabilities, percentiles, and inverse calculations
  • t-Distribution - Critical values, confidence intervals, and p-values
  • F-Distribution - ANOVA, variance ratio tests, and critical values
  • Chi-Square Distribution - Goodness of fit, variance tests, and critical values
  • Exponential & Gamma - Waiting times and reliability analysis
  • Beta & Uniform - Bounded continuous distributions

Discrete Distributions

Calculators for discrete probability distributions:

  • Binomial Distribution - Success/failure trials, exact and cumulative probabilities
  • Poisson Distribution - Rare events, count data, rate calculations
  • Hypergeometric Distribution - Sampling without replacement
  • Negative Binomial - Number of trials until r successes
  • Geometric Distribution - Trials until first success

Test Hypothesis Distributions

Comprehensive hypothesis testing tools:

  • t-Tests - One-sample, two-sample (independent & paired), Welch's t-test
  • ANOVA - One-way, two-way, repeated measures, and post-hoc tests
  • Chi-Square Tests - Independence, goodness of fit, homogeneity
  • Proportion Tests - One proportion, two proportions, multiple proportions
  • Correlation Tests - Pearson, Spearman, partial correlation
  • Non-parametric Tests - Mann-Whitney, Kruskal-Wallis, Wilcoxon, Friedman

Call Center Staffing Calculator

Optimize staffing levels for call centers and service operations:

  • Erlang C Calculator - Queue waiting times, service level calculations
  • Staffing Optimization - Determine optimal number of agents for target service levels
  • Monte Carlo Simulation - Model complex arrival patterns and service variations
  • Performance Metrics - Average speed of answer (ASA), abandonment rates, occupancy
  • Scenario Analysis - Compare different staffing strategies and schedules
  • Cost Analysis - Balance service quality with staffing costs

Power & Sample Size Calculator

Determine optimal sample sizes and statistical power using advanced methods:

  • t-Tests Power Analysis - One-sample, two-sample, paired comparisons
  • ANOVA Power Analysis - F-tests with non-central F distributions
  • Non-Central Distributions - Accurate power calculations for t and F tests
  • Correlation & Regression - Power for correlation coefficients and R²
  • Proportion Tests - Sample size for one and two proportion tests
  • Interactive Visualization - Power curves, effect size relationships, sample size trade-offs

Quality Control Calculator

Statistical process control and quality management tools:

  • Control Charts - X-bar, R, S, p, np, c, u charts with control limits
  • Process Capability - Cp, Cpk, Pp, Ppk indices and sigma levels
  • Acceptance Sampling - Single, double, and sequential sampling plans
  • Six Sigma Tools - DPMO, sigma level conversions, process yield
  • Measurement System Analysis - Gage R&R, repeatability and reproducibility
  • Pareto Analysis - Identify and prioritize quality issues
