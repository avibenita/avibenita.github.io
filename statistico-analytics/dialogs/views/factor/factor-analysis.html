Structure Overview
Rotation: iChoose how factors are rotated after extraction. Varimax (orthogonal) maximises variance of squared loadings per factor, keeping factors uncorrelated. Promax and Oblimin allow factors to correlate (oblique), often yielding cleaner patterns. None uses unrotated components.
Simple Structure: iOverall cleanliness of the loading pattern: Excellent when every variable loads on exactly one factor, degrading as cross-loadings accumulate. ...
Cross-loadings: iNumber of variables that load meaningfully (above the cutoff) on more than one factor simultaneously. High counts suggest factors are not cleanly separated. ...
Loading Cutoff: iMinimum absolute loading used to classify a variable as associated with a factor. Loadings below this value are treated as negligible cross-loadings. ...
Factor Correlation Max: iLargest absolute correlation between any two factors (only relevant for oblique rotations). Values below 0.30 suggest factors are nearly orthogonal and Varimax may be preferable. ...

 Loading Highlight Rules

      
How cell colour and weight are determined in the Rotated Loadings Matrix

      


      

        

          
        

        

          
Threshold (slider)

          
Only loadings with an absolute value ≥ the slider value receive colour and weight. Everything below is greyed-out at 38% opacity to suppress visual noise from near-zero loadings.

        

      


      

        

          
        

        

          
Colour intensity

          
Within the highlighted range, brightness scales linearly with how far a loading exceeds the threshold:

          intensity = ( |loading| − threshold ) / ( 1 − threshold )
          
A loading just above the threshold is faint; a loading of 1.0 is fully saturated.

        

      


      

        

          
        

        

          
Colour direction

          

            ● Blue  =  positive loading

            ● Amber  =  negative loading


            Both use the same intensity formula — only the hue differs.
          

        

      


      

        

          
        

        

          
Bold weight

          
A loading becomes bold (700) when its intensity is ≥ 0.6 — meaning it sits in the top 40 % of the highlighted range. Below that it uses semi-bold (600).

          

            Example: threshold = 0.30, loading = 0.72

            → intensity = (0.72 − 0.30) / (1 − 0.30) = 0.60 → bold ✓
