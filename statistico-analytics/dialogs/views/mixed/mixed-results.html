    Estimation & Convergence Parameters controlling how the mixed model was fitted. REML is the default method; the optimizer and stopping criteria determine whether a stable solution was reached.
    Method Estimation method. REML (Restricted Maximum Likelihood) gives unbiased variance-component estimates but cannot compare models with different fixed effects. ML (Maximum Likelihood) is required for LRT comparisons across fixed-effect structures.
    —
    Optimizer Numerical algorithm used to maximise the (restricted) log-likelihood. Common choices: nlminb (L-BFGS-B variant, fast), bobyqa (derivative-free, robust for complex random structures), nloptwrap (nonlinear optimisation wrappers).
    —
    Convergence Whether the optimizer reached a stable solution. Failure to converge indicates the model may be over-parameterised (too many random effects for the sample size) or that the data contain near-collinear predictors. Try a simpler random structure or a different optimizer.
    —
    Iterations Number of optimizer cycles before convergence (or the iteration limit). A large number relative to the default ceiling may indicate a flat or poorly-conditioned likelihood surface.
    —
    DF Method Method used to approximate degrees of freedom for fixed-effect tests. Satterthwaite approximates df from the gradient of the variance matrix (fast). Kenward-Roger applies a bias-corrected covariance matrix (more accurate for small samples and complex random structures, but slower).
    —
    Cov. Structure Assumed structure of the residual covariance matrix (R-matrix). VC (Variance Components, default): diagonal — residuals are independent and identically distributed. AR(1): residuals within a group follow a first-order autoregressive process. CS: compound symmetry — constant correlation between all pairs within a group. UN: fully unstructured — each pair has its own covariance parameter.
    —
    Information Criteria Model-fit indices based on the log-likelihood penalised for complexity. Used to compare competing models fitted to the same data — lower is better. They do not test significance; they measure relative fit.
    Criterion All criteria are derived from −2LL (deviance). They differ only in how they penalise the number of estimated parameters (p) and sample size (n). Prefer AIC for prediction, BIC for parsimony when n is large. Value Absolute values are not interpretable in isolation. Compare the same criterion across models fitted to identical data. A difference of ≥ 2 AIC/BIC units is considered meaningful. Formula & Penalty

    How to use: Lower values indicate better fit when comparing competing models fitted to the same dataset. A difference of ≥ 2 units is considered meaningful. AIC favours predictive accuracy; BIC penalises complexity more strongly and is preferred for parsimony.
    Likelihood Ratio Test (vs. Intercept-Only) Tests whether the full model (fixed + random effects) fits significantly better than a baseline model with only an intercept. The test statistic is Δ(−2LL) ~ χ² with Δdf degrees of freedom. Important: LRT requires ML estimation, not REML.
    Model Model being compared. The intercept-only (null) model has no predictors — only a grand mean and the random structure. The full model adds all specified fixed effects. −2LL Minus twice the log-likelihood (deviance). Lower = better fit. The LRT statistic is the difference between the two −2LL values. Params Total number of estimated parameters including fixed effects, variance components, and covariance parameters. The difference in params (Δdf) is the chi-square test degrees of freedom. Δ−2LL Difference in −2LL between this model and a simpler (nested) model. Distributed as χ² with Δdf degrees of freedom under H₀. Larger values indicate a greater improvement in fit. Δdf Degrees of freedom for the chi-square test = difference in number of parameters between the two models. Must be ≥ 1. χ² Sig. p-value for the LRT. Small p (< .05) means the more complex model fits significantly better than the simpler one. Requires ML (not REML) when comparing models with different fixed effects.

    R²m is the variance explained by fixed effects alone. R²c includes both fixed and random effects. The difference (R²c − R²m) is the variance explained by clustering. The ICC measures how similar observations within the same group are.

    Fixed Effects Contribution Relative contribution of each fixed-effect predictor based on the absolute standardised t-value. This is an indicator of importance, not a formal partial R². Larger bars indicate stronger evidence for that predictor's effect.

    Covariance Structure Specifies the assumed structure of the R-matrix (residual covariance within groups) and the G-matrix (random-effects covariance). The default VC structure assumes independent residuals. More complex structures (AR1, CS, UN) model within-group correlation patterns for longitudinal or repeated-measures data.
    Estimated Residual Covariance Matrix (Σ) The R-matrix (Σ). Describes the covariance structure of residuals within groups. For VC (default) this is σ²·I (diagonal). For AR(1) or CS it captures within-subject correlations across time points.

    G-Matrix — Random Effects Covariance The G-matrix contains the variances and covariances of the random effects. The diagonal elements are random-effect variances; off-diagonal elements are covariances between random slopes/intercepts.

    Covariance Structure Comparison (AIC) Side-by-side AIC comparison across covariance structures fitted to the same data. Lower AIC = better balance of fit and parsimony. Select the structure with the lowest AIC — but only choose a more complex structure if ΔAIC ≥ 2 and the structure is theoretically justified (e.g. AR(1) for time-series).
    Forest Plot — Fixed Effect Estimates (95% CI) Each row shows a predictor's point estimate (B) with its 95% confidence interval. The vertical reference line at 0 marks no effect. Intervals that do not cross zero are statistically significant at α = .05. Width of the interval reflects estimation precision — wider intervals indicate more uncertainty, typically due to small sample size or high collinearity.
