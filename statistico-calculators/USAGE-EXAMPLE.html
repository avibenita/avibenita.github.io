Calculator Integration Examples

This page demonstrates how to integrate the Sample Size Calculator into other modules.

Example 1: ANOVA Power Button

Click the button to navigate to the calculator with ANOVA parameters pre-filled:

navigateToCalculator('anova', {
    numGroups: 3,
    effectSizeF: 0.25,
    alpha: 0.05,
    power: 0.80
});

Example 2: One-Sample t-test

navigateToCalculator('one-sample-mean', {
    effectSize: 0.5,
    alpha: 0.05,
    power: 0.80,
    alternative: 'two-sided'
});

Example 3: Using URL in href (for links)

Open ANOVA Calculator (Link) 
<a href="#" onclick="window.location.href = getCalculatorURL('anova', {
    numGroups: 4, 
    effectSizeF: 0.3, 
    alpha: 0.05, 
    power: 0.90
}); return false;">
    Open Calculator
</a>

Example 4: Open in New Window

navigateToCalculator('regression', {
    numPredictors: 2,
    rsquared: 0.15,
    alpha: 0.05,
    power: 0.80
}, true, true);  // autoCalculate=true, newWindow=true

Available Test Types

  • 'one-sample-mean' - One-Sample t-test
  • 'two-sample-mean' - Two-Sample t-test
  • 'one-sample-proportion' - One-Sample Proportion
  • 'two-sample-proportion' - Two-Sample Proportions
  • 'correlation' - Correlation Analysis
  • 'anova' - One-Way ANOVA
  • 'regression' - Linear Regression

Parameter Reference

Common Parameters (all tests):

  • alpha - Significance level (default: 0.05)
  • power - Statistical power (default: 0.80)
  • alternative - 'two-sided' or 'one-sided'

Test-Specific Parameters:

  • ANOVA: numGroups (or k), effectSizeF (or f)
  • One-Sample Mean: effectSize (or d)
  • Two-Sample Mean: effectSize (or d), allocationRatio
  • Proportions: p1, p2 (for two-sample)
  • Correlation: rho (or r), rho0 (or r0)
  • Regression: numPredictors (or p), rsquared (or r2 or R2)
