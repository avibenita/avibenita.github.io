Install Statistico to open the Calculator Hub

The Hub opens as an app, not inside a regular browser tab. Install it once, then open Statistico from your apps list or home screen.

1
Install Statistico
Use the button below. If your browser does not show a popup, use the menu in the top-right and choose Install app.
2
Close this browser tab
After the install finishes, do not continue in this tab. This page is only for installation.
3
Open the app
Launch Statistico from Start Menu, Apps list, or Home Screen to access the Hub.
Ready to install?

Installation help

Manual installation in Chrome

Use these steps only if the install button does not open a browser prompt.

  1. Open the browser menu (⋮) in the top-right corner.
  2. Choose Install Statistico or Save and share → Install page as app.
  3. Finish the installation.
  4. Close this tab.
  5. Open Statistico from your apps list or home screen.
