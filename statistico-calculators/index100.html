Fast path (least confusing)

  1. If you see Open in app in Chrome address bar, click it.
  2. If Chrome menu has no Install app, the app is already installed.
  3. To make it visible in Start menu: open chrome://apps, right-click Statistico Calculator Hub, choose Create shortcuts..., and enable Start menu.
  4. After that, open from Windows Start by searching Statistico.

If Start still does not show it: press Win+R, run shell:AppsFolder, then pin Statistico Calculator Hub to Start.
